#Assignment a04: Use a new package lubridate to wrangle the date data and find the total number of birds by park and month

#Introduction: Ecological datasets are often stored across multiple tables that must be combined before meaningful analysis can occur. In thie exercise, I practiced using tidy data principles and data wrangling tools in R to combine and restructure bird observation datasets. The goal was to integrate information about bird observations and parks, then summarize bird counts across selected parks and months. This workflow demonstrates how tidy data manipulation allows researchers to transform raw observational data into more interpretable summaries.

#Data: The analysis uses bird observation data containining information on survey dates, park locations, species observed, and the number of birds counted. The dataset includes variables such as survey_date, park_name, common_name, and bird_count. To prepare the data for analysis, the survey date was used to create a new variable representing the month of each observation. The data were then filtered to include only four parks of interest: Lindo, Orme, Palomino, and Sonrise.

#Analysis:Data wrangling was perfomed using functions from the tidyverse. First, the month of each observation was extracted from the survey date using the lubridate::month() function and converted to a factor variable. The new survey_month column was relocated next to the survey date to improve readability. The dataset was then filtered to include only observations from the selected parks. Finally, the data were grouped by park and survey month, and the total number of birds observed was calculated using summarize(). This produced a tidy summary table showing bird counts by location and month.
